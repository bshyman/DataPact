## Installation
Add this line to your application's Gemfile:

```ruby
gem "data_pact"
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install data_pact
```

Mount the engine in your `config/routes.rb` file:
```ruby
mount DataPact::Engine => "/data_pact"
```
